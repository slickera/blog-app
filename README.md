# Blog App

A full-stack blog application, built from scratch as a learning project.

**Status:** Work in progress

## Stack

- **Frontend:** React + Vite
- **Backend:** Node.js + Express
- **Database:** PostgreSQL + Prisma ORM
- **Auth:** JWT stored in httpOnly cookies

## Running locally

```bash
# Backend
cd server
npm run dev

# Frontend
cd client
npm run dev
```
